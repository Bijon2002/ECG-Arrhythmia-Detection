🫀 Uncertainty-Aware ECG Arrhythmia Detection

Using Deep Ensembles — B.Sc. (Hons) Computer Science and Software Engineering

Project Overview & Problem Statement

Problem Statement: Traditional Deep Learning models for ECG analysis suffer from overconfidence, providing "confident" but incorrect predictions on complex or noisy arrhythmias, which can mislead medical professionals.
Solution: This project introduces an Uncertainty-Aware clinical decision support system. It leverages True Deep Ensembles and novel uncertainty quantification (such as Cluster-Based Entropy) to accurately report both the diagnosis and the AI's confidence, ensuring physicians know exactly when to trust the AI and when manual review is required.

Architecture & Methodology

1. Deep Ensembles

3 Independent 1D CNN models trained from scratch to capture diverse feature representations.

2. AAMI Standards

Trained on 100,000+ beats from the MIT-BIH Database, categorized into 5 clinical classes.

3. Uncertainty Metrics

MC Dropout: Epistemic
Predictive Entropy: Total Doubt
Cluster Entropy: Medical Context

4. AI Calibration

Temperature Scaling applied to logits to align probabilities with true correctness.
